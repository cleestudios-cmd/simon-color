How this works

  • Arrays — The computer stores the color sequence in an array (sequence). Each round adds one random number (0–3) to the end.
  • Event listeners — Buttons use click and pointerdown so taps and clicks run the same game logic.
  • Async timing — The sequence plays one color at a time using async/await and short delays, so the screen stays responsive.
  • Game loop — Each round: add a color → flash the sequence → wait for your taps → if correct, score up and repeat; if wrong, game over.
  • localStorage — Your name, personal best, and the leaderboard are saved as JSON in the browser. Refreshing the page keeps the same list (stored on this device).
  • Leaderboard — Top scores are sorted highest first. Gold, silver, and bronze highlight places 1–3.
